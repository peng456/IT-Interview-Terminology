# HR专业术语

1.HC  headcount         指公司HR(人力资源)预计招聘的员工人数,遇到有人说锁HC

2.JD  job description   工作岗位描述  （这里不是指京东）

3.base                   指基础工资，无责任底薪

4.职级   阿里  P序列=技术岗P4~P14  M序列=管理岗    滴滴 D5~D13   腾讯 T1.x~T4.y    百度 T1~T8  
(参考https://www.zhihu.com/question/19683731 很全)
